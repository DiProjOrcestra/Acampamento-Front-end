📜 O Chamado dos Três Livros

Esta jornada é dividida em três grandes tomos de poder. Cada um deles é obrigatório quer crescer como dev no mercado:

  • Livro 1 – HTML: O Primeiro Poder
    Forja a estrutura das páginas, a base de todo castelo digital.
    👉 Entrar no Acampamento de HTML
  • Livro 2 – CSS: A Magia Visual
    Veste o site com a identidade do reino, cores, fontes e layouts responsivos.
    👉 Desbloquear a Magia do CSS
  • Livro 3 – JavaScript: A Magia Viva
    Dá vida às páginas: interações, formulários, efeitos e lógica de negócios.
    👉 Despertar o Poder do JavaScript

Ao concluir os três livros, você estará pronto para:

  • Entender de verdade como cada página do site funciona por baixo dos panos.
  • Criar novas seções, landing pages e componentes sem depender de plugins.
  • Dar o próximo passo na trilha: React e front-end profissional.

🛡️ Como conquistar esta saga

Cada capacitação foi pensada como uma campanha com níveis, XP e desafios práticos. Não é teoria solta: você vai construir coisas de verdade que poderão ser usadas no seu portfólio, na empresa e na sua carreira.

  • Siga a ordem: primeiro HTML, depois CSS, depois JavaScript.
  • Faça os desafios, não só leia. O XP só conta quando você pratica.
  • Guarde seus arquivos de treino: eles serão base para estudos futuros (React, APIs, etc.).
