📜 O Chamado dos Quatro Livros

Esta jornada foi expandida. Para se tornar um arquiteto de sistemas lendário e dominar o desenvolvimento em equipe, você deverá percorrer quatro grandes tomos de poder:

  • Livro 0 – GitHub: O Grimório
    Aprenda a controlar o tempo, trabalhar em branches e compartilhar código com a Guilda.
    👉 Abrir o Grimório de Versionamento
  • Livro 1 – HTML: O Primeiro Poder
    Forja a estrutura das páginas, a base de todo castelo digital.
    👉 Entrar no Acampamento de HTML
  • Livro 2 – CSS: A Magia Visual
    Veste o site com a identidade do reino, cores, fontes e layouts responsivos.
    👉 Desbloquear a Magia do CSS
  • Livro 3 – JavaScript: A Magia Viva
    Dá vida às páginas: interações, formulários, efeitos e lógica de negócios.
    👉 Despertar o Poder do JavaScript

Ao concluir a saga completa, você estará pronto para:

  • Versionar projetos em equipe usando Conventional Commits e Pull Requests.
  • Entender de verdade como cada página do site funciona por baixo dos panos.
  • Criar novas seções e componentes sem depender de construtores visuais.

🛡️ Como conquistar esta saga

Cada capacitação foi pensada como uma campanha com níveis, XP e desafios práticos. Não é teoria solta: você vai construir o seu Portfólio do Herói, versioná-lo no GitHub e deixá-lo dinâmico.
