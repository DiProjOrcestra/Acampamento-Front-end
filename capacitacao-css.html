📖Prólogo: Invocando a Magia das Folhas de Estilo

Se HTML é o pergaminho onde a história é escrita, CSS são as tintas, o brasão do reino, o brilho das armaduras e o fogo das tochas na parede do castelo.

CSS (Cascading Style Sheets) é a linguagem usada para estilizar e organizar o layout de páginas web: cores, fontes, espaçamentos, tamanhos, colunas, sombras e até animações.

Importante: Em todo este livro você vai trabalhar em cima do mesmo portfolio.html criado no treinamento de HTML, adicionando um arquivo style.css para aplicar o tema medieval da marca.

💡O que é CSS e como funciona

CSS em uma frase

CSS é uma linguagem que diz ao navegador como mostrar o HTML: quais cores usar, qual fonte, qual tamanho, quanto espaço entre elementos e como posicionar tudo na tela.

🪄Nível 1: Invocando CSS (inline, interno e externo)

Três formas de lançar magia CSS

Inline: direto na tag HTML (útil para testes, ruim para projetos grandes).

<p style="color: #2EB600; font-size: 20px;">
  Texto em verde
</p>

CSS interno: dentro da tag <style> no <head>.

<head>
  <style>
    p {
      color: #01571E;
    }
  </style>
</head>

CSS externo (forma ideal): arquivo separado style.css.

<head>
  <link rel="stylesheet" href="style.css">
</head>
/* style.css */
body {
  background-color: #FFFBDC;
  font-family: Arial, sans-serif;
}

🎯 Missão: Ligando o Portfólio ao CSS

  1. Na mesma pasta do portfolio.html, crie um arquivo style.css.
  2. No <head> do portfolio.html, adicione: 
    <link rel="stylesheet" href="style.css">
  3. No style.css, defina:
    • Cor de fundo do body usando a paleta da marca.
    • Fonte padrão do site.
    • Uma cor para o <header> e o <footer>.

⚠️ Armadilhas Comuns

  • Salvar como style.css.txt em vez de .css.
  • Errar o caminho em href do <link>.
  • Esquecer chaves {} ou ponto-e-vírgula ;.
🏆 Recompensa: +30 XP de Conjuração de Estilo.

🎯Nível 2: Seletores e Especificidade

Selecionando alvos para sua magia

Para aplicar CSS, é preciso selecionar os elementos HTML corretos.

/* Seletor por tag */
p {
  color: #003D0D;
}

/* Seletor por classe */
.card-projeto {
  background-color: #FFFFFF;
  border: 1px solid #01571E;
}

/* Seletor por id */
#painel-heroi {
  border: 2px dashed #2EB600;
}

Use class para estilos que se repetem e id para algo único na página.

Combinando seletores

/* Links dentro do nav do portfólio */
nav a {
  color: #01571E;
  text-decoration: none;
}

🎯 Missão: Destacando Seções do Portfólio

  1. No portfolio.html, garanta que seções tenham ids: #sobre, #projetos, #contato, #painel-heroi.
  2. No style.css, crie estilos para cada seção (fundos levemente diferentes, espaçamento, etc.).
  3. Crie uma classe .botao-primario para estilizar botões com as cores da marca e aplique ao botão do formulário.
🏆 Recompensa: +40 XP de Mira Seletora.

🎨Nível 3: Cores, Fontes e Texto

Cores – tingindo o portfólio

Você pode usar nomes, rgb(), hex ou hsl().

body {
  background-color: #FFFBDC;
  color: #003D0D;
}

header, footer {
  background-color: #01571E;
  color: #FFFBDC;
}

a {
  color: #2EB600;
}

Fontes e tipografia

body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  line-height: 1.5;
}

h1, h2, h3 {
  font-family: 'Cinzel', serif;
}

Você pode importar fontes do Google Fonts via <link> ou @import.

Outras propriedades de texto

#sobre p {
  text-align: justify;
}

#projetos h2 {
  text-transform: uppercase;
  letter-spacing: 1px;
}

🎯 Missão: Deixando o Portfólio Legível e Bonito

  1. Defina uma fonte agradável para o texto geral e uma mais “medieval” para títulos.
  2. Use cores da paleta para links e títulos.
  3. Ajuste line-height para melhorar a leitura do texto “Sobre mim”.
🏆 Recompensa: +40 XP de Carisma Visual.

🛡️Nível 4: Box Model – a armadura dos elementos

Entendendo a caixa

Todo elemento é uma caixa com conteúdo, padding, borda e margem; isso é o box model. 

#sobre, #projetos, #contato, #painel-heroi {
  padding: 20px;
  margin: 20px 0;
  border-radius: 8px;
  background-color: #FFFFFF;
}

box-sizing: border-box

Com border-box, largura/altura incluem padding e borda, facilitando layout responsivo. 

* {
  box-sizing: border-box;
}

🎯 Missão: Emoldurando Seções do Portfólio

  1. Use padding para criar “respiro” interno nas seções do portfólio.
  2. Use margin para separar as seções entre si.
  3. Adicione borda ou sombra leve em #projetos e #painel-heroi para destaque.
🏆 Recompensa: +60 XP de Defesa Layout.

📦Nível 5: Display, Layout Básico e Flexbox

display e layout básico

header nav ul {
  list-style: none;
  display: flex;
  gap: 16px;
  justify-content: center;
}

Use display: flex para organizar itens de navegação e cards de projetos.

Projetos em cards

.lista-projetos {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}

.card-projeto {
  flex: 1 1 250px;
  padding: 12px;
  border-radius: 6px;
  border: 1px solid #01571E;
  background-color: #FFFFFF;
}

🎯 Missão: Layout do Portfólio

  1. Transforme a lista de projetos em cards usando flexbox.
  2. Faça o header com logo/nome e navegação em linha usando flex.
  3. Garanta que o layout funcione bem em telas médias (tablet).
🏆 Recompensa: +70 XP de Arquitetura de Layout.

📱Nível 6: Responsividade e Media Queries

O que é responsividade?

Um site responsivo se adapta a diferentes telas (celular, tablet, desktop) usando unidades flexíveis e media queries.

Media queries aplicadas ao portfólio

@media (max-width: 768px) {
  header nav ul {
    flex-direction: column;
  }

  .lista-projetos {
    flex-direction: column;
  }
}

Isso significa: “se a largura da tela for menor que 768px, reorganize navegação e projetos em coluna”.

🎯 Missão: Portfólio que funciona no celular

  1. Adicione uma media query para reorganizar o header e os projetos em coluna no mobile.
  2. Ajuste fontes e espaçamentos para telas pequenas.
  3. Teste no celular ou no modo responsivo do DevTools.
🏆 Recompensa: +75 XP de Adaptação Móvel.

🐉Boss Final: Portfólio do Herói (HTML + CSS)

Missão: Vestir o Portfólio do Herói

Agora você vai transformar o portfolio.html criado no Livro 1 em um site completo visualmente, usando a paleta da marca e tudo que aprendeu de CSS.

Requisitos mínimos

  • Usar o mesmo portfolio.html da capacitação de HTML (não criar outro arquivo).
  • Manter um único style.css bem organizado (por seções ou por tipo de componente).
  • Aplicar a paleta da marca em:
    • Header, footer e botões principais.
    • Links, destaques e cartões de projeto.
  • Criar layout agradável para:
    • Seção “Sobre mim” com imagem e texto lado a lado (em telas maiores).
    • Lista de projetos em cards com flexbox.
    • Formulário de contato com inputs alinhados.
    • “Painel do Herói” com caixa destacada (que será interativa com JS depois).
  • Tornar o portfólio responsivo em telas pequenas (uma coluna, fontes legíveis).

🎯 Boss Mission (Livro 2)

  1. Refine o layout do portfolio.html até ficar apresentável como seu cartão de visitas.
  2. Use apenas as cores definidas no manual da marca nesta etapa.
  3. Garanta que tudo continue legível e utilizável em celular.
  4. Deixe IDs e classes prontos para o JavaScript (por exemplo #hero-xp, .btn-xp, etc.).
🏆 Recompensa Final: +120 XP – Seu portfólio agora tem uma armadura visual completa.
