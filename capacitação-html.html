💡Conceitos Fundamentais: A Lore da Web

Como a Web Funciona (Resumido)

Imagina que você é um cliente em um restaurante (seu navegador) e o chef é um servidor:

  1. Você faz um pedido → Seu navegador envia um pedido para um servidor.
  2. Chef prepara a comida → O servidor processa e monta a resposta.
  3. Comida chega na mesa → Você recebe HTML, CSS, JavaScript, imagens.
  4. Você vê o resultado → Seu navegador desenha tudo na tela.

A Tríade Sagrada do Web Dev

Tecnologia O Que Faz Analogia
HTML Define ESTRUTURA (títulos, textos, imagens, links) Esqueleto do corpo humano
CSS Define APARÊNCIA (cores, fontes, posicionamento) Roupa e maquiagem
JavaScript Define COMPORTAMENTO (cliques, validações, animações) Músculos e movimento

Navegador: Seu Intérprete Mágico

Seu navegador lê um arquivo .html e desenha na tela. Teste isto AGORA:

  1. Abra qualquer site (Google, por exemplo).
  2. Pressione F12 ou Ctrl+Shift+I (Windows) ou Cmd+Option+I (Mac).
  3. Você verá o código HTML da página! Tudo que você vê foi criado assim.
🎯 Desafio Conceitual:
  1. Abra o Developer Tools de um site que você gosta.
  2. Encontre uma tag <h1> (título grande).
  3. Veja o HTML por trás daquilo que você vê.

🔥Nível 1: HTML Básico – Seu Primeiro Documento

Conceito: Tags e Elementos

HTML funciona com tags. Uma tag é um rótulo que diz ao navegador "isso é um parágrafo", "isso é um título", etc.

Estrutura de uma tag:

<nomeDaTag>Conteúdo</nomeDaTag>

Exemplos:

<h1>Isto é um Título Grande</h1>
<p>Isto é um parágrafo normal</p>
<a href="https://google.com">Isto é um link</a>

REGRA OURO: Toda tag que abre deve fechar!

O Documento HTML Mínimo

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título da Minha Página</title>
</head>
<body>
  <!-- Seu conteúdo vai aqui -->
</body>
</html>

🎯 Desafio Prático: Seu Primeiro Site

  1. Abra seu editor de texto (VS Code).
  2. Crie um novo arquivo chamado index.html.
  3. Cole o código acima.
  4. Salve (Ctrl+S).
  5. Abra no seu navegador (arraste o arquivo ou clique com botão direito → Abrir com).

VEJA A MAGIA ACONTECER! ✨

⚠️ Armadilhas Comuns

  • ❌ Esqueci de fechar a tag: Toda tag aberta precisa fechar com </nomeDaTag>.
  • ❌ Salvei em .txt: O arquivo precisa terminar em .html.
  • ❌ Não consigo ver a página: Arraste o arquivo correto para o navegador.
🏆 Recompensa: ✅ 25 XP – Seu primeiro documento HTML!

🏰Nível 2: Tags Importantes – Construindo a Estrutura

Tags de Texto: Títulos

<h1>Título Mega Grande (use UMA vez por página)</h1>
<h2>Subtítulo</h2>
<h3>Menor ainda</h3>
...
<h6>O menor título</h6>

Parágrafos e Quebras

<p>Isto é um parágrafo.</p>
<p>Este é outro parágrafo separado.</p>
<br> <!-- Quebra de linha -->
<hr> <!-- Linha horizontal -->

Formatação de Texto

<strong>Muito importante (negrito)</strong>
<em>Ênfase (itálico)</em>
<u>Sublinhado</u>
<del>Deletado (tachado)</del>
<small>Texto pequeno</small>

🎯 Desafio Prático: Página Estruturada

Crie nivel2.html com:

  • Um título h1.
  • Alguns parágrafos com formatação.
  • Pelo menos um h2 e h3.
  • Quebras de linha.

🔗Nível 3: Links e Imagens – Conectando o Mundo

Links (`<a>`)

<!-- Link para site externo -->
<a href="https://google.com">Ir para Google</a>

<!-- Link para outra página -->
<a href="pagina2.html">Ir para Página 2</a>

<!-- Link para seção específica -->
<a href="#secao-importante">Ir para Seção</a>

<!-- Abrir em nova aba -->
<a href="https://google.com" target="_blank">Google (nova aba)</a>

Imagens (`<img>`)

<!-- Imagem local -->
<img src="gato.jpg" alt="Um gato fofo">

<!-- Imagem de URL -->
<img src="https://example.com/imagem.png" alt="Exemplo">

<!-- Com tamanho -->
<img src="gato.jpg" alt="Gato" width="300" height="200">
Atributo ALT é essencial! Aparece se a imagem não carregar e é importante para acessibilidade.

🎯 Desafio Prático: Links e Imagens

Crie nivel3.html com:

  • Pelo menos 3 links (internos e externos).
  • Pelo menos 2 imagens.
  • Use https://via.placeholder.com/150 para imagens de teste.
🏆 Recompensa: ✅ 45 XP – Você conecta o mundo!

📋Nível 4: Listas e Tabelas – Organizando Dados

Listas Não-Ordenadas

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Listas Ordenadas

<ol>
  <li>Primeiro</li>
  <li>Segundo</li>
  <li>Terceiro</li>
</ol>

Tabelas

<table border="1">
  <tr>
    <th>Nome</th>
    <th>Nível</th>
  </tr>
  <tr>
    <td>Você</td>
    <td>1</td>
  </tr>
</table>

🎯 Desafio Prático: Listas e Tabelas

  • Crie uma lista não-ordenada de itens.
  • Crie uma lista ordenada (passos).
  • Crie uma tabela com pelo menos 3 colunas e 4 linhas.
🏆 Recompensa: ✅ 55 XP – Seus dados estão organizados!

📝Nível 5: Formulários – Capturando Informação

Estrutura Básica

<form action="processar.php" method="POST">
  <!-- Campos do formulário -->
</form>

Tipos de Input

<label for="nome">Seu Nome:</label>
<input type="text" id="nome" name="nome" placeholder="Digite aqui">

<input type="email" name="email" placeholder="seu@email.com">
<input type="password" name="senha">
<input type="number" name="idade" min="1" max="120">

<select name="classe">
  <option>Selecione</option>
  <option value="guerreiro">Guerreiro</option>
</select>

<textarea name="msg" rows="4" cols="30"></textarea>

<button type="submit">Enviar</button>
IMPORTANTE: Sempre use <label> para acessibilidade!

🎯 Desafio Prático: Formulário Completo

Crie um formulário com:

  • Campos de texto, email, número.
  • Select dropdown.
  • Textarea.
  • Checkbox ou radio.
  • Botões submit e reset.
🏆 Recompensa: ✅ 65 XP – Você captura informações!

🐉Boss Final: Portfólio do Herói (HTML para ser evoluído com CSS e JS)

Missão: Criar o Portfólio do Herói

Este será o seu projeto central ao longo da jornada. Agora você monta a estrutura em HTML; nas próximas capacitações, vai deixar essa página bonita com CSS e interativa com JavaScript.

Requisitos do Projeto (HTML)

  • ✅ Arquivo principal chamado portfolio.html.
  • ✅ Usar estrutura HTML semântica.
  • ✅ Incluir seções:
    • <header> com seu nome e navegação.
    • <section id="sobre"> – “Sobre Mim”.
    • <section id="projetos"> – “Meus Projetos”.
    • <section id="contato"> – com formulário simples.
    • <section id="painel-heroi"> – “Painel do Herói” (ficha que será animada depois com JS).
  • ✅ Adicionar pelo menos 1 imagem sua (ou avatar) e links para redes/portfólio.
  • ✅ Usar classes e IDs que serão úteis depois no CSS/JS (por exemplo: .card-projeto, #xp-hero).
